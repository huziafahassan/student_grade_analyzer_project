# student_grade_analyzer_project

## Description

The **Student Grade Analyzer** is a beginner-friendly Python project that calculates and analyzes the academic performance of a student. The program prompts the user to enter the student's name, the total number of subjects, the maximum marks per subject, and the marks obtained in each subject.

Based on the entered data, the program calculates the **total obtained marks**, **total marks**, **percentage**, and assigns a **letter grade** according to predefined grading criteria. It then displays a clear and organized result summary.

This project demonstrates the use of fundamental Python concepts, including:

* User input and output
* Variables and arithmetic operations
* Loops (`for`)
* Conditional statements (`if-elif-else`)
* Formatted output using f-strings

## Features

* Enter student name
* Input marks for multiple subjects
* Calculate total and obtained marks
* Compute percentage automatically
* Assign grades based on percentage
* Display a structured result summary

This project is designed for beginners who want to strengthen their understanding of Python programming by building a practical, real-world application.
